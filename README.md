## 🎯 项目 1：积分平台 BI 仪表盘

**位置**：`积分平台BI仪表盘/`

**简介**：基于 Python Flask + ECharts + SQL Server 的数据可视化大屏，深度蓝赛博立体风格，展示商家、会员、礼品、订单 4 大主题的 BI 分析。

### 技术栈

| 层 | 技术 |
|----|------|
| 后端 | Python 3.12 + Flask 3.1 |
| 数据库 | SQL Server 2019+ (BIDemo_AccumulateCoin) |
| 驱动 | pyodbc 5.2 + ODBC Driver 17 |
| 仪表盘 | ECharts 5.4 + 中国地图 GeoJSON |
| ETL | 10 个 SQL 存储过程 |

### 核心功能

- **6 块 KPI** + **8 大区块**（地图/趋势/分类/组合图/榜单/仪表/订单流）
- **7 个子页面**（商家/会员/ETL/报表/告警/系统 + 详情）
- **20 个 REST API**（13 仪表盘 + 7 子页面）
- **深蓝赛博立体视觉**（深黑 + 冰蓝 + 紫色辅助）
- **真实数据接入**（13 商家 / 5 万会员 / 5 万交易流水 / 800 订单）

### 启动

```bash
cd 积分平台BI仪表盘
start.bat    # 双击或命令行启动
# 浏览器访问 http://127.0.0.1:5000
```

**依赖**：先执行 `sql/00_complete_init.sql` 创建数据库

### 详细文档

参见 `积分平台BI仪表盘/README.md`

## 🎯 项目 2：毕设项目（业务系统）

**位置**：`毕设项目/`

**简介**：积分兑换平台的**业务后台系统**，包含 ASP.NET WebForms 旧版本（业务系统/）+ Flask 新版本（jinbi_app/），实现会员/商家/积分码/订单/兑换/购物车等业务流程。

### 子项目

#### 2.1 业务系统（ASP.NET）

**位置**：`毕设项目/业务系统/`

- **技术**：ASP.NET WebForms + C# + SQL Server
- **页面**：会员注册/登录、修改个人信息、礼品、购物车、首页等
- **资源**：20+ 张图片资源（1.jpg ~ JBI.jpg + ZS.png）
- **项目文件**：`业务系统.sln` + `.csproj` 多个

#### 2.2 jinbi_app（Flask）

**位置**：`毕设项目/jinbi_app/`

- **技术**：Flask + Python
- **入口**：`app.py` (23KB)
- **启动**：`start.bat` / `stop.bat` / `_test_start*.bat`
- **依赖**：`.venv/`（虚拟环境）

### 业务闭环

```
会员注册
  ↓
购买商品获得积分码
  ↓
商家发放 / 平台发放
  ↓
消费者兑换礼品
  ↓
生成订单 / 配送 / 完成
```
